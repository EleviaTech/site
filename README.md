# Elévia Tech V2

Versão experimental da landing/portfolio da Elévia Tech.

## O que esta V2 inclui

- Hero tecnológico com canvas animado e mockup de dashboard.
- Seletor de negócio com solução dinâmica por segmento.
- Portfólio vivo com link para a demo de estética publicada.
- Simulação de atendimento inteligente.
- Diagnóstico rápido com recomendação e link para WhatsApp.
- Assistente virtual local, sem backend, pronto para GitHub Pages.

## Como abrir

Abra `index.html` no navegador ou rode um servidor local:

```bash
python3 -m http.server 4174
```

Depois acesse `http://localhost:4174`.
